# Hand Gesture Recognition with CNN

This project uses a Convolutional Neural Network (CNN) to recognize 10 different hand gestures from images.

## Classes
- Palm
- l
- Fist
- Fist_moved
- Thumb
- Index
- Ok
- Palm_moved
- C
- Down

## Files
- `gesture_train.ipynb`: Model training and evaluation
- `hand_gesture_Model.keras`: Saved trained model
